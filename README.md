# Capacitor App

This is a [Capacitor](https://capacitor.ionicframework.com/) app. It's a simple app that shows how to use Capacitor and Laravel.

## Frontend Ionic

Install Ionic CLI:
```bash
npm install -g @ionic/cli
```

```bash
cd capacitor-ionic-fe
npm i
ionic cap run
```

## Frontend Capacitor

```bash
cd capacitor-fe
npm i
npm start
```

## Backend

```bash
cd capacitor-be
composer install
npm i
composer dev
```
